    MapSchema.PutPartial (map_schema v0.1.0)

    The PutPartial module compone the macros that let us build the put/2 that put a new values usign a map.

    It´s method check the type of every property following the schema.

    Example:

    person = Person.new()
  |> Person.put(%{"name" => "ric",  "age"=> 29})

assert Person.get_name(person) == "ric"
assert Person.get_age(person) == 29
